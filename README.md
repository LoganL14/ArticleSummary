# Automating the collection of Arxiv Articles 

## Overview
This project automates the collection of scientific articles from https://arxiv.org, gets PDFs and Markdown files to feed into LLMs.

### Article Collection
- Query arXiv API for articles from two days ago (UTC).
- Parse XML response using BeautifulSoup.
- Extract PDF links and download articles into a structured folder system.
